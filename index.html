選課怎麼選

操作路徑與流程

  1. 學校首頁 → I-TOUCH → 選課系統 → 以 CYCU Myself 帳號登入。
  2. 先用「預排功課表」規劃，確認時段沒有衝堂（7/15 起已開放）。
  3. 正式選課分兩軌開放：「一般 / 教育學程」與「通識 / 體育 / 軍訓」，兩邊時間不同，兩軌都要記得選。
  4. 選課結果不滿意可在「加退選」階段調整，通識加退選採先選先上。

容易忽略的規則

  • 通識別只搶熱門課，天、人、物、我四學類要平均分配，不要卡在同一類。
  • 部分通識基礎必修需擇一：「宗教信仰與靈性關懷」／「生命意義與價值判斷」擇一；「公民素養與社會關懷」擇一；「歷史思維與多元文化」擇一。
  • 「宗教哲學」與「人生哲學」不可同學期選修。
  • 每學期至多選 2 門通識延伸選修課程。

通識課程學分結構

類別規則學分
通識基礎必修天、人、物、我四學類皆須修習指定必修課16
通識延伸選修天、人、物、我每學類至少 2 學分，其餘 4 學分自由選12
合計（畢業門檻）110 學年度後入學適用28

四學年分開修，大一不用一次修完——但建議每學期至少接觸 1–2 個學類，避免大三、大四才擠在一起搶課。
